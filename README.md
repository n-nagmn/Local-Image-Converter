# Local Image Converter

ブラウザ完結で動作する、完全ノーログの画像変換ツールです。サーバーへのデータ送信を一切行わず、クライアントサイドのJavaScriptのみで画像の変換やリサイズ処理を完結させます。

## 特徴

* **完全ノーログかつオフライン動作**: アップロードした画像は外部サーバーに一切送信されません。
* **多彩なフォーマット対応**: JPG, PNG, WEBP, AVIF, BMP, ICO, GIF, PDFへの変換をサポートします。
* **一括変換とZIPダウンロード**: 複数ファイルをまとめて変換し、1つのZIPファイルとして一括保存可能です。
* **リサイズ機能**: ピクセル単位で幅や高さを指定してリサイズできます。片方のみ指定した場合はアスペクト比を自動で維持します。
* **リアルタイムサムネイル**: 変換対象の画像をリスト上でプレビューできます。
